# May the force be with VUE

## Project setup

Install dependencies

```
npm install
```

Compiles and hot-reloads for development

```
npm run serve
```

Compiles and minifies for production

```
npm run build
```

Run tests

```
npm run test
```

## Docker

Build image

```
docker build . -t [tag]
```

Run image

```
docker run -p 8080:8080 [tag]
```

## Lint and Prettier

Lint all files

```
npm run lint
```

Prettier all files

```
npm run prettier
```
